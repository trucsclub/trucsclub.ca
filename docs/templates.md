# Templates & example files

This page lists the example/template files available under `docs/templates/`. Use these as copy-paste starters or references.

- [`docs/templates/club-main.json`](docs/templates/club-main.json) — Copy-paste-ready `main.json` template (recommended for club pages). Paste into `src/lib/data/pages/<slug>/main.json`.
- [`docs/templates/clubs-example.json`](docs/templates/clubs-example.json) — Example structure for `src/lib/data/clubs.json` showing available club fields and shapes (not used by the site; for reference).
- [`docs/templates/projects-example.json`](docs/templates/projects-example.json) — Example structure for `src/lib/data/projects.json` showing project shapes.

Notes:
- Files ending in `-example.json` are templates only and are not consumed by the site; they demonstrate the available fields and shapes.
- For the field-by-field reference see [docs/data-attributes.md](docs/data-attributes.md).
