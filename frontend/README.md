## Initializing repo

```sh
git remote add origin https://github.com/Aidanjosiah02/trucomputing.ca.git
# fetch and switch to main.
nvm install latest
nvm use latest
npm install
# Then switch to your desired branch.

```

## start development server:

```sh
npm run dev
```

## Create production version:

```sh
# preview
npm run preview
#build
npm run build
```
